What to pay attention to when drawing a portrait

There are a few important items to consider when trying to make a portrait realistic. You will want to spend a lot of time on the following:

  • Face shape
  • Facial Proportions
  • Shading/light source

If you don't put a lot of care into the above 3 items, your drawing can easily start to look strange. Compare with your reference photo often to make sure you're being accurate.

Basic Steps

You can follow these steps in order to complete a portrait:

  1. Sketch face shape -- Use light pencils so you can correct mistakes!
  2. Sketch facial features
  3. Compare with reference to make sure your sketch is accurate before continuing
  4. Using darker pencils, add shading to the facial features
  5. Shade the skin and hair
  6. Go over drawing with blending tool and/or eraser to make adjustments
